## üßÆ Example Algorithm

### Binary Search

- **File**: `binarySearch.js`
- **Description**: Performs binary search on a sorted array to find the index of the target element.
- **Usage**:
    ```javascript
    const result = binarySearch([1, 2, 3, 4, 5, 6], 4);
    console.log(result); // Output: 3
    ```
